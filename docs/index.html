dotfiles is my dotfiles for UNIX.

This repository is inspired by github.com/haru067/dotfiles .

Installation

curl -fsSL https://dot.usagiga.me/install.sh | sh

NOTE : dotfiles creates symlinks. It can't replace symlinks if there're file or dir already.
